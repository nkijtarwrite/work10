# 西式甜點商場

這是一套以純 PHP 實作的期末專題範例，主題為「西式甜點商場」。專案以單一前端入口搭配 JSON 儲存層完成，適合在一般 PHP 主機或 XAMPP 環境中直接展示。

## 功能

- 會員註冊、登入、登出與帳號啟用信件流程
- 商品上架、圖片上傳、編輯與刪除
- 顧客瀏覽商品、加入購物車、預約取貨與下單
- 店家後台管理商品、訂單與會員
- 訂單統計圖表與銷售摘要

## 專案主題

主題設定為「西式甜點商場」，模擬一個提供精緻甜點販售與預訂取貨的網站。顧客可以瀏覽甜點、加入購物車、選擇取貨日期並送出預約；店家管理員則可以管理商品、訂單、會員與營收圖表。

## 目前的實作方式

- 後端：純 PHP
- 儲存：JSON 檔案
- 圖表：Chart.js
- 風格：自訂甜點商場 UI

## 預設管理員

- 信箱：`admin@dessert.local`
- 密碼：`Admin1234!`

## 執行方式

若本機已安裝 PHP，可在專案根目錄執行：

```bash
php -S localhost:8000 -t public
```

然後打開 `http://localhost:8000`。

## 主要資料檔

- `data/users.json`：會員資料
- `data/products.json`：商品資料
- `data/orders.json`：訂單資料
- `data/outbox/`：啟用信測試內容

## 目錄結構

- `app/`：共用工具與資料存取
- `public/`：前端入口、靜態資源與上傳圖片
- `data/`：JSON 儲存資料

## 教學重點

這份專題可以作為期末報告時的說明基礎：

- 登入前與登入後的不同介面
- 顧客下單流程與訂單建立邏輯
- 管理員如何維護商品、會員與訂單
- 如何從訂單資料產生統計圖表
